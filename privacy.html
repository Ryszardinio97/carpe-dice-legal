🎲 Carpe Dice — Polityka Prywatności

Ostatnia aktualizacja: 5 lipca 2026

1. Administrator danych

Administratorem danych osobowych jest twórca aplikacji Carpe Dice, dostępny przez platformę Discord.

2. Jakie dane zbieramy

Dane Źródło Cel
Nick Discord (nazwa użytkownika) Discord OAuth2 API Wyświetlanie w grze i rankingach
Discord ID (unikalny identyfikator) Discord OAuth2 API Identyfikacja gracza w bazie rankingów
Wyniki rozgrywek (punkty) Generowane w grze Rankingi graczy

3. Czego NIE zbieramy

  • Adresu e-mail
  • Hasła
  • Danych płatniczych
  • Danych lokalizacyjnych
  • Treści prywatnych wiadomości Discord

4. Jak przechowujemy dane

Dane rankingowe (nick, Discord ID, punkty) przechowywane są w bazie danych PostgreSQL na serwerach Railway.app. Dane są chronione standardowymi mechanizmami bezpieczeństwa.

5. Udostępnianie danych

Nie sprzedajemy ani nie udostępniamy danych osobowych podmiotom trzecim. Nick i wyniki rankingowe są widoczne publicznie w tabeli rankingów w aplikacji.

6. Integracja z Discord

Aplikacja korzysta z Discord OAuth2 wyłącznie w celu pobrania nicku i ID użytkownika. Nie przechowujemy tokenów dostępu. Więcej informacji o polityce prywatności Discord: discord.com/privacy.

7. Prawa użytkownika

Masz prawo do:

  • Wglądu w swoje dane przechowywane w rankingach
  • Żądania usunięcia swoich danych — skontaktuj się przez Discord

8. Pliki cookie

Aplikacja nie używa plików cookie.

9. Zmiany polityki

Zastrzegamy sobie prawo do zmiany niniejszej Polityki Prywatności. O istotnych zmianach poinformujemy przez aktualizację daty na tej stronie.

10. Kontakt

W sprawie danych osobowych skontaktuj się przez Discord.
